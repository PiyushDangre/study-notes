JVM - Java Virtual Machine is just a specifictation for the provision and working of JRE which would run byte code compiled from Java files. 
JRE - Java Runtime Environment is the actual `implementation` of the JVM Specification. It is physically present. It is collection of libraries and other files which the JVM uses at runtime. 
JDK - Java Development Kit is the combination of JRE + Development tools required for creating Java applications. It has additional components such as Java Compiler (javac), Code loader/Interpreter, Archiver (JAR) and Documentation (javadoc) etc.
