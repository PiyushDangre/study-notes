### Encapsulation

- This is concept for just binding the data members and methods together as a single unit.
- data members of a class will be hidden from other classes -> Accessible only through methods of that class.
- That's why called as **__Data Hiding__**.
- Keep variables private and methods public.
- We can create read-only classes (by removing setters) and write only classes (by removing getters) - using encapsulation concept.
